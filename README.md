# Short North Studio Homepage

This package is the Wix-free homepage for Short North Studio.

## Upload instructions

1. In GitHub, create a new **public** repository named `short-north-studio`.
2. Open the repository and choose **Add file → Upload files**.
3. Upload `index.html` and `styles.css` from this folder.
4. Click **Commit changes**.
5. Open **Settings → Pages**.
6. Under **Build and deployment**, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
7. Click **Save**.

The temporary site address will be:

`https://info-beep.github.io/short-north-studio/`

## Custom domain

After the temporary site works:

1. In GitHub: **Settings → Pages → Custom domain**
2. Enter `short-north-studio.com`
3. Save.
4. Update the domain DNS records to GitHub Pages.
5. Enable **Enforce HTTPS** after GitHub verifies the domain.
